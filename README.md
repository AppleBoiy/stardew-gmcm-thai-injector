**มีคำแปลให้เลือก 2 สไตล์:**
1. **สไตล์เกมเมอร์ (กาวๆ):** แปลแบบกันเอง อ่านสนุก มีศัพท์แสงเกมเมอร์
2. **สไตล์ทางการ (ปกติ):** แปลแบบตรงไปตรงมา เป็นระเบียบ อ่านแล้วเข้าใจง่าย

1. ดาวน์โหลดไฟล์สคริปต์ `inject_manifest_descriptions.command`
2. ดับเบิลคลิกที่ไฟล์เพื่อรันสคริปต์
   * **สำหรับ Mac/Linux:** หากเปิดไม่ได้ ให้เปิด Terminal แล้วพิมพ์ `chmod +x inject_manifest_descriptions.command` เพื่ออนุญาตให้รันได้
   * **สำหรับ Windows:** สามารถรันผ่าน Command Prompt ได้โดยพิมพ์ `python inject_manifest_descriptions.command`
3. สคริปต์จะค้นหาโฟลเดอร์ `Mods` ในเครื่องของคุณอัตโนมัติ (หากหาไม่เจอ จะมีช่องให้ลากโฟลเดอร์ Mods มาวาง)
4. รอจนกว่ามันจะสแกนและอัปเดตคำอธิบายเสร็จเรียบร้อย!

### ⚠️ ข้อควรระวัง
* สคริปต์นี้จะทำการ **เขียนทับ** ไฟล์ `manifest.json` ของม็อดต่างๆ โดยตรง
* หากคุณทำการอัปเดตม็อดเหล่านั้นผ่าน SMAPI หรือโหลดเวอร์ชันใหม่มาลง คำอธิบายภาษาไทยอาจจะหายไป ให้กลับมารันสคริปต์นี้ใหม่อีกครั้ง
* ปลอดภัย 100% ไม่มีไวรัส เป็นเพียงสคริปต์ Python ที่ใช้แทนที่ข้อความ (Regex Replace) ธรรมดาครับ
